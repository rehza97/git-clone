POST /v1/projects

Create a Project

This endpoint initiates the archiving process for a new software project. It accepts metadata about the project and returns a unique project ID along with an upload URL for the source code archive.

info
Compliance Note

Projects must adhere to Decision 1275 guidelines. Ensure all contributors are listed correctly.

Request Body

Parameter Type Description
name string The official name of the project. *Required
version string Semantic versioning (e.g., 1.0.0).
contributors array List of researcher IDs or emails.
license string SPDX license identifier (e.g., MIT, Apache-2.0).
Request 
curl -X POST https://api.ascap.dz/v1/projects \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "AI Traffic Control System",
    "version": "1.0.0",
    "description": "An AI model for optimizing traffic flow in Algiers.",
    "license": "MIT",
    "contributors": [
      "researcher_12345", 
      "researcher_67890"
    ],
    "tags": ["AI", "Smart City", "Transport"]
  }'
Response (201 Created) 
{
  "id": "proj_8923fd2",
  "status": "pending_upload",
  "upload_url": "https://storage.ascap.dz/upload/v1/signed/...",
  "expires_at": "2023-10-25T14:30:00Z",
  "created_at": "2023-10-25T13:30:00Z"
}
GET /v1/projects/{id}/certificate

Download Certificate

Retrieve the official PDF certificate of deposit for an archived project. This document serves as proof of registration for academic defenses.

Path Parameters

Parameter Type Description
id string The unique identifier of the project.
Request 
curl -X GET https://api.ascap.dz/v1/projects/proj_8923fd2/certificate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -o certificate.pdf
Response (200 OK)
< Binary PDF data stream >
GET /v1/search

Search Archive

Search the public archive for projects. Useful for discovering existing solutions and verifying project uniqueness.

Query Parameters

  • q (string): The search query.
  • university (string): Filter by university affiliation.
  • year (integer): Filter by academic year.
Request 
import requests
response = requests.get(
    "https://api.ascap.dz/v1/search",
    params={
        "q": "machine learning", 
        "university": "USTHB"
    },
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)
print(response.json())
Response (200 OK) 
{
  "count": 14,
  "results": [
    {
      "id": "proj_123",
      "name": "ML for Image Proc",
      "author": "Yacine K."
    }
  ]
}
