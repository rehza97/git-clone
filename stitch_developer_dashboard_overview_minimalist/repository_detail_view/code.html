AI Analysis Module - Algerian Steppe

This repository contains the source code for the "Agri-Steppe" project, aimed at optimizing agricultural yield in semi-arid zones using satellite imagery and local sensor data.

Project Goals (Decision 1275)

  • Analyze soil moisture trends over 10 years.
  • Predict drought risks using LSTM models.
  • Provide actionable insights for local farmers.

Installation

git clone https://ascap.dz/usthb/ai-analysis.git
cd ai-analysis
pip install -r requirements.txt

About

Research project for Master's Thesis at USTHB. Focus on deep learning applications in North African climate.
